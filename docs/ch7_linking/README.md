# #7 링커

## 이 장에서 다루는 내용
프로그램을 실행 파일로 만들기 위해 여러 오브젝트 파일을 하나로 연결하는 **링킹(Linkinkg)**의 개념을 학습한다. </br>
정적 링크, 동적 링크, 심볼 해결, 재배치 등 다양한 링킹 기술과 함께 링커의 역할을 이해하는 것이 목표이다.

> 실행 파일이 만들어지는 마지막 단계에서 어떤 일이 일어나는지를 알아보자.
