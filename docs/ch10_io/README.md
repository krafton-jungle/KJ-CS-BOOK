# #10 시스템 수준 I/O


## 이 장에서 다루는 내용
프로그램에서 사용되는 파일 입출력의 개념을 학습한다.
Unix I/O 시스템, RIO 패키지의 필요성, 파일 메타데이터 조회, I/O 재지정 등을 이해하는 것이 목표이다. 

> "유닉스에서는 모든 것이 파일이다"라는 말을 이해해보자. 

---

## 학습 목표
- UNIX I/O 시스템과 이점 이해
- 기본 시스템콜(open, close, read, write) 이해
- 네트워크 통신의 경우 Rio 패키지를 사용하는 것이 필요함을 이해
- 파일 디스크립터 테이블, 오픈 파일 테이블, v-node 테이블 이해
- stat() 라이브러리 함수 사용법 익히기 
- dup2() 시스템콜의 사용법과 동작과정 이해
